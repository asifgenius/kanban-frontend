## Installation

First create .env accroding to .env.example. Make sure you have added backend url.

#### Node Server

```sh
$ npm install
```

Run the server project
```sh
$ npm start
```

Verify the deployment to you server address in your browser
```sh
$ 127.0.0.1:3000
```

Build the server project
```sh
$ npm run build
```
